## Tips for development

Install test tools

    npm install

Prepare target directory for node-red

    mkdir -p ~/.node-red
    cd ~/.node-red
    npm install ~/src/node-red-contrib-interpolate/ --save

Run node-red

    npm run nodered

Open http://127.0.0.1:1880/ in browser
